# 图表（chart）

图表组件支持在卡片中展示多种类型的统计图表，包括折线图、柱状图、饼图等，用于数据可视化展示。

## 功能说明

图表组件支持以下图表类型：
- 折线图 (line)
- 柱状图 (bar)
- 饼图 (pie)
- 环形图 (ring)
- 进度条 (progress)
- 雷达图 (radar)
- 散点图 (scatter)

## JSON 结构

```json
{
  "schema": "2.0",
  "body": {
    "elements": [
      {
        "tag": "chart",
        "chart_type": "line",
        "height": "200px",
        "x_field": "time",
        "y_field": "value",
        "series_field": "category",
        "data": [
          {"time": "1月", "value": 120, "category": "产品A"},
          {"time": "2月", "value": 200, "category": "产品A"},
          {"time": "3月", "value": 150, "category": "产品A"},
          {"time": "1月", "value": 100, "category": "产品B"},
          {"time": "2月", "value": 180, "category": "产品B"},
          {"time": "3月", "value": 220, "category": "产品B"}
        ],
        "options": {
          "title": {
            "text": "月度销量统计",
            "show": true
          },
          "legend": {
            "show": true
          },
          "smooth": true,
          "colors": ["#165DFF", "#00B42A"]
        }
      }
    ]
  }
}
```

## 字段说明

| 名称 | 必填 | 类型 | 默认值 | 描述 |
|------|------|------|--------|------|
| tag | 是 | String | / | 组件的标签，固定为 `chart` |
| chart_type | 是 | String | / | 图表类型，可选值：`line`(折线图)、`bar`(柱状图)、`pie`(饼图)、`ring`(环形图)、`progress`(进度条)、`radar`(雷达图)、`scatter`(散点图) |
| height | 否 | String | `200px` | 图表高度，单位为px，范围 [100, 1000]px |
| x_field | 否 | String | / | x轴字段名，适用于折线图、柱状图等二维图表 |
| y_field | 否 | String | / | y轴字段名，适用于折线图、柱状图等二维图表 |
| series_field | 否 | String | / | 系列字段名，用于多系列图表 |
| data | 是 | Array | / | 图表数据源，数组格式，每个元素为一个数据对象 |
| options | 否 | Object | / | 图表配置选项，根据不同图表类型有不同配置 |

## 不同图表类型示例

### 折线图示例

```json
{
  "tag": "chart",
  "chart_type": "line",
  "height": "250px",
  "x_field": "month",
  "y_field": "sales",
  "data": [
    {"month": "1月", "sales": 120},
    {"month": "2月", "sales": 200},
    {"month": "3月", "sales": 150},
    {"month": "4月", "sales": 280},
    {"month": "5月", "sales": 220}
  ],
  "options": {
    "smooth": true,
    "point": {
      "show": true
    },
    "label": {
      "show": true
    }
  }
}
```

### 柱状图示例

```json
{
  "tag": "chart",
  "chart_type": "bar",
  "height": "250px",
  "x_field": "product",
  "y_field": "sales",
  "data": [
    {"product": "产品A", "sales": 280},
    {"product": "产品B", "sales": 350},
    {"product": "产品C", "sales": 190},
    {"product": "产品D", "sales": 420}
  ],
  "options": {
    "color": ["#165DFF"]
  }
}
```

### 饼图示例

```json
{
  "tag": "chart",
  "chart_type": "pie",
  "height": "250px",
  "data": [
    {"type": "分类A", "value": 30},
    {"type": "分类B", "value": 25},
    {"type": "分类C", "value": 20},
    {"type": "分类D", "value": 25}
  ],
  "options": {
    "angleField": "value",
    "colorField": "type",
    "radius": 0.8,
    "label": {
      "type": "outer"
    }
  }
}
```

## 注意事项

1. 图表数据量建议不超过100条，过多数据可能导致渲染性能问题
2. 不同图表类型支持的配置选项不同，具体请参考飞书开放平台完整文档
3. 图表组件仅支持展示功能，不支持交互操作
4. 为保证显示效果，建议图表高度不低于150px
