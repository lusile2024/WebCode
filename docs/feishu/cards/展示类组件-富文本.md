# 富文本（Markdown）

富文本组件支持完整的 Markdown 语法，用于展示格式化的文本内容，支持列表、表格、代码块等复杂排版。

## 功能说明

富文本组件相比于普通文本的 lark_md 类型，支持更丰富的 Markdown 语法，适合展示复杂的文档内容、帮助说明、公告等信息。

## JSON 结构

```json
{
  "schema": "2.0",
  "body": {
    "elements": [
      {
        "tag": "markdown",
        "content": "# 标题\n\n这是一段**富文本**内容，支持 Markdown 语法。\n\n- 列表项1\n- 列表项2\n- 列表项3\n\n| 列1 | 列2 |\n|-----|-----|\n| 单元格1 | 单元格2 |\n\n```javascript\nconsole.log('Hello World');\n```\n",
        "text_size": "normal",
        "text_align": "left"
      }
    ]
  }
}
```

## 字段说明

| 名称 | 必填 | 类型 | 默认值 | 描述 |
|------|------|------|--------|------|
| tag | 是 | String | / | 组件的标签，固定为 `markdown` |
| content | 是 | String | / | Markdown 格式的文本内容 |
| text_size | 否 | String | `normal` | 文本大小，可选值同普通文本组件 |
| text_align | 否 | String | `left` | 文本对齐方式，可选值：`left`、`center`、`right` |

## 支持的 Markdown 语法

### 基础语法

| 语法 | 效果 |
|------|------|
| `# 一级标题` | 一级标题 |
| `## 二级标题` | 二级标题 |
| `### 三级标题` | 三级标题 |
| `**粗体**` | **粗体** |
| `*斜体*` | *斜体* |
| `~~删除线~~` | ~~删除线~~ |
| `[链接文字](https://example.com)` | [链接文字](https://example.com) |
| `<at id=all></at>` | @所有人 |
| `<at id=user_id></at>` | @指定用户 |
| `:emoji_code:` | 飞书表情 |

### 列表

**无序列表：**
```markdown
- 列表项1
- 列表项2
  - 子列表项1
  - 子列表项2
```

**有序列表：**
```markdown
1. 第一项
2. 第二项
3. 第三项
```

### 表格

```markdown
| 列标题1 | 列标题2 | 列标题3 |
|---------|---------|---------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

### 代码块

```markdown
```javascript
function hello() {
  console.log('Hello World');
}
```
```

### 引用

```markdown
> 这是一段引用文本
> 支持多行引用
```

### 分隔线

```markdown
---
```

## 注意事项

1. 富文本组件不支持嵌入图片、视频等多媒体内容
2. 复杂的表格建议控制列数，避免在移动端显示溢出
3. 代码块会自动换行，适合展示代码片段
4. 支持的 Markdown 语法子集可能会随着飞书版本更新而扩展，建议使用基础语法保证兼容性
