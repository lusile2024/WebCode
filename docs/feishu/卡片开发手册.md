# 飞书卡片开发核心手册

## 已获取的核心文档
### ✅ 卡片JSON 2.0版本更新说明
完整内容已保存为：[卡片JSON-v2变更说明.md](./cards/卡片JSON-v2变更说明.md)

**核心要点：**
1. **结构变更**：新增`body`字段，所有`elements`必须放在`body`下
2. **必填字段**：`schema: "2.0"`、`config.update_multi: true`（暂不支持false）
3. **按钮交互**：`value`字段必须是JSON字符串，不再支持对象
4. **组件变更**：废弃`note`组件和`action`模块，使用`div`+按钮替代
5. **布局优化**：容器间距默认值调整，新增`extra_large`间距选项
6. **富文本**：支持标准Markdown语法，新增`<link>`标签支持多端跳转

## 快速参考
### 卡片V2标准结构
```json
{
  "schema": "2.0",
  "config": {
    "update_multi": true,
    "enable_forward": true
  },
  "header": {
    "template": "blue",
    "title": {
      "tag": "plain_text",
      "content": "卡片标题"
    }
  },
  "body": {
    "elements": [
      // 卡片内容组件
    ]
  }
}
```

### 按钮标准格式
```json
{
  "tag": "button",
  "text": {
    "tag": "plain_text",
    "content": "按钮文本"
  },
  "type": "primary",
  "value": "{\"action\": \"action_name\", \"param\": \"value\"}"
}
```

> ⚠️ 注意：`value`必须是JSON序列化后的字符串，不能直接传对象
