## 4. 常见问题排查

### 问题1: 机器人没有回复
**可能原因**:
- 飞书应用配置不正确
- 机器人没有权限在群聊中发言
- WebSocket连接失败

**排查步骤**:
1. 检查应用日志中的错误信息
2. 确认飞书应用的"机器人与应用"功能已启用
3. 确认机器人已添加到群聊中
4. 尝试私聊机器人

---

### 问题2: 卡片显示格式错误
**可能原因**:
- 飞书卡片JSON格式不正确
- 飞书卡片schema版本不匹配

**排查步骤**:
1. 查看应用日志中的卡片JSON输出
2. 验证卡片JSON是否符合飞书v2.0规范
3. 检查特殊字符是否正确转义

---

### 问题3: 命令列表为空
**可能原因**:
- FeishuCommandService初始化失败
- 命令加载异常被静默处理

**排查步骤**:
1. 查看应用日志中的Warning/Error
2. 确认FeishuCommandService已正确注册
3. 检查是否有"加载插件命令失败"的日志
